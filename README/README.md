# Bank Loan Excel Dashboard

## Project Overview

This project presents an interactive Bank Loan Dashboard built in Microsoft Excel to analyze lending performance, loan portfolio health, borrower characteristics, and repayment trends.

The dashboard enables users to monitor key performance indicators (KPIs), analyze loan distributions, and make data-driven business decisions using interactive slicers and visualizations.

---

## Dashboard Pages

- Summary Dashboard
- Overview Dashboard
- Details Dashboard

---

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting

---

## Key KPIs

- Total Loan Applications
- Total Funded Amount
- Total Amount Received
- Average Interest Rate
- Average DTI
- Good Loan vs Bad Loan Analysis
